В этом репозитории находится набор инструментов для автоматического тестирования вашего процессора на Verilog. В отличие от unit-тестов, которые проверяют отдельные модули, эти тесты запускают полноценные ассемблерные программы, проверяя работу всего процессора целиком (Control Unit, ALU, память, переходы).

**Весь код тестов написан с помощью gemini 3 pro и незначительно уточнен автором.** 

За сим, принимаются улучшения, дополнения, исправления. С этими целями вы можете написать мне [@da1grn](https://t.me/da1grn). В том числе, если вы хотели бы видеть какой нибудь класс тестов, но не хотите писать его вручную, то можете не стесняться написать мне.

В данный момент, реализация автора проходит все тесты. В случае, если у вас тесты не проходят, попробуйте проверить свой код на надежность перед тем как писать мне.
